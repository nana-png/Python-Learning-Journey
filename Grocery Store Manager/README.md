# 🛒 Grocery Store Manager

A simple command-line grocery management application built with Python.

This project allows users to manage a grocery list through an interactive menu system. It was built to practice Python fundamentals such as lists, sets, loops, conditionals, functions, and user input.

---

## ✨ Features

- 📋 View all grocery items
- ➕ Add new items
- ❌ Remove items
- 🔍 Search for an item
- 🛍️ Display unique items using Python sets
- 📊 Show the total number of items
- 🔤 Display items that start with a chosen letter
- 📑 Automatically numbers and sorts the grocery list

---

## 🛠 Technologies Used

- Python 3

## 🚀 How to Run

```bash

```
